## 1. 초기 포트폴리오 냉정한 평가

### 🔍 종합 평가: C+ ~ B- 수준

**문제점 요약:**
- SV 포지션 지원 시 채용 가능성: **거의 제로**
- 경력의 일관성 부족으로 신뢰도 하락
- 본부 경험이 전혀 없는데 본부 관점으로 포장
- 검증 불가능한 수치와 과대 포장

## 2. 치명적인 약점 분석

### ❌ 약점 1: 경력의 일관성 부족 (가장 큰 문제)

**경력 타임라인:**
```
2014-2017: 인테리어 과장 (3년)
2017-2018: 파리바게트 직원 (1.5년) ← 왜 과장에서 직원으로?
2018-2022: 개인카페 운영 (3.5년) ← 매출/수익 데이터 전무
2022-2023: 메가커피 매니저 (6개월) ← 너무 짧음
2023-2025: 자활센터 팀장 (2년) ← 일반 프랜차이즈와 괴리
2025.09: 해운회사 프로젝트 (실패) ← 최근 실패 케이스
```

**인사권자가 보는 것:**
- 인테리어 과장 → 파리바게트 직원: **경력 역주행**
- 개인카페 3.5년 후 메가커피 6개월: **개인카페 실패로 추정**
- 자활센터 2년: **일반 수익 중심 프랜차이즈 경험 아님**
- 해운회사 프로젝트 무산: **비즈니스 판단력 의문**

### ❌ 약점 2: 프랜차이즈 본부 실무 경험 제로

**현재 강조하는 "SV 역량"의 문제:**
- ✗ 실제 SV로 일한 경험 없음
- ✗ 다점포 관리 경험 없음
- ✗ 가맹점 정기 점검 실무 경험 없음
- ✗ 본부 시스템 구축/운영 경험 없음

**결론:**
> 이건 SV가 아니라 "매장 운영자"입니다.

### ❌ 약점 3: 수치와 성과의 신뢰성 문제

**초기 SSOT vs 현재 사이트 비교:**
- SSOT 초안: "매출 150% 회복", "30% 단축", "200% 증가"
- 현재 사이트: 수치 전부 삭제 → 정성적 표현으로 변경

**문제점:**
- 검증 가능한 수치를 스스로 뺐다 = **증명 못함**
- "신입 교육 2주→5일"도 검증 방법 없음
- "배민 매출 150% 회복"도 before/after 데이터 없음

### ⚠️ 과대 포장된 부분

#### 1. "10년 경력" 표현
- F&B 7년 + 공간설계 3년 = 10년이라고 표기
- **실제 F&B**: 파리바게트 1.5년 + 개인카페 3.5년 + 메가커피 0.5년 + 자활센터 2년 = 7.5년
- 공간설계 3년은 F&B와 다른 분야인데 합산 표기 → **과장**

#### 2. "프랜차이즈 솔루션" 6개
- 실제로는 개인 경험을 "본부 관점"으로 포장한 것
- 한 번도 본부에서 일한 적 없는데 "본부가 얻는 가치" 제시
- 이론과 가정일 뿐, **검증된 솔루션 아님**

#### 3. AI/자동화 부분
- n8n 워크플로우, ChatGPT 활용은 좋음
- 하지만 **프랜차이즈 본부에서 이걸 원하나?**
- 본부는 ERP, POS, 발주 시스템 이미 있음
- 개인 생산성 도구와 본부 시스템은 **다른 차원**

### 1️⃣ 가맹점 교육 담당 (★★★★★ 90%)

**왜 이게 제일 맞냐면:**

✅ **당신의 진짜 강점:**
- 커피 이론 교육자료 **47페이지** 직접 제작
- 신입 교육 체계 구축 경험 (자활센터)
- 추출 매뉴얼, 체크리스트 제작
- 실제 현장에서 신입 교육해본 경험
- 대형(메가커피) + 개인(씨리얼) 양쪽 경험

✅ **프랜차이즈 본부가 원하는 것:**
- 신규 가맹점주 교육 자료 제작
- 오픈 전 직원 교육 진행
- 매뉴얼 지속 업데이트
- 본부 교육장에서 실습 진행

**구체적 직무:**
- 신규 가맹점주 대상 초기 교육 (2-3일)
- 가맹점 직원 교육 커리큘럼 제작
- 품질 관리 체크리스트 제작/배포
- 온라인 교육 콘텐츠 제작 (영상, PPT)

**예상 연봉:**
- 중소 프랜차이즈: 3,000~3,500만원
- 대형 프랜차이즈: 3,500~4,000만원

**어필 포인트:**
```
"자활센터에서 2년간 신입 교육 체계를 구축했습니다.
- 커피 이론 교육자료 47페이지 제작
- 신입 적응 기간 2주→5일로 단축
- 에스프레소 추출 매뉴얼로 맛 편차 제거

이 경험을 OO브랜드 전 가맹점에 적용하여
교육 효율을 높이고 싶습니다."
```

**왜 이것도 잘 맞냐면:**

✅ **당신의 배경:**
- 인테리어 3년 (현장 감리, 견적 협의)

- 신규 가맹점 오픈 전 현장 지원
- 인테리어 업체 미팅 동석 (본부 입장 대변)
- 오픈 전 점검 (설비, 동선, 메뉴 세팅)
- 오픈 초기 안정화 지원 (1-2주)

**구체적 직무:**
- 신규 점포 오픈 전 체크리스트 검수
- 인테리어 시공 감리 (본부 기준 준수 확인)
- 오픈 전 시연 교육 및 최종 점검
- 오픈 초기 1-2주 현장 상주 지원

**예상 연봉:** 3,200~3,800만원 (출장 많음)

**어필 포인트:**
```

해운회사 카페 프로젝트에서는
- 상권 분석 20곳 벤치마킹

- 메뉴 구성 및 동선 설계

이 경험으로 신규 가맹점의 성공적인 오픈을 지원하겠습니다."
```

### 4️⃣ 중소 프랜차이즈 초기 멤버 (ALL-IN-ONE) (★★★★☆ 70%)

**이게 사실 제일 현실적일 수도:**

**대상:**
- 가맹점 10~30개 규모 중소 프랜차이즈
- 본부 인원 5~10명

**역할:**
- 교육 + 오픈 지원 + 콘텐츠 + 매뉴얼 **전부**
- "운영지원팀" 이름이지만 실제론 **만능 실무자**

**✅ 왜 여기가 좋냐:**
- 큰 회사는 분업화되어 경쟁 심함
- 작은 회사는 "다재다능한 현장형 인재" 필요
- 당신처럼 이것저것 해본 사람이 딱 맞음

**예상 연봉:** 2,800~3,500만원 (작은 회사라 낮음, 대신 성장 가능성)

**장점:**
- 다양한 업무 경험 가능
- 초기 멤버로 성장 시 핵심 인력
- 본부 경험 쌓기 좋음

**단점:**
- 급여 낮음
- 업무 강도 높음 (뭐든 해야 함)
- 회사 망하면 끝
