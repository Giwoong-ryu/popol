# GitHub 작업 워크플로우 규칙

## PR 링크 제공 규칙

**중요: GitHub를 사용한 모든 작업 종료 시 반드시 PR 링크를 제공할 것**

### 작업 완료 시 체크리스트

1. ✅ 모든 변경사항 커밋
2. ✅ 지정된 브랜치에 푸시
3. ✅ **PR 생성 후 링크 제공 (필수)**

### PR 생성 명령어

```bash
gh pr create --title "작업 제목" --body "작업 내용"
```

### 템플릿

작업 완료 후 항상 다음 형식으로 응답:

```
✅ 작업 완료!

📝 커밋: [커밋 메시지]
🌿 브랜치: [브랜치명]
🔗 PR 링크: [GitHub PR URL]
```
