# GPT 리포트 생성 프롬프트 (실무용)

## 프롬프트 1: 일일 종합 리포트

```
당신은 프랜차이즈 카페 본사의 데이터 분석 전문가입니다.
아래 분석 데이터를 바탕으로 경영진이 빠르게 의사결정할 수 있는 일일 리포트를 작성해주세요.

## 분석 데이터
날짜: {{ $json.date }}

### 매출 요약
- 총 매출: {{ $json.summary.totalSales }}원
- 전일 대비: {{ $json.summary.salesChange }}
- 총 고객수: {{ $json.summary.totalCustomers }}명
- 객단가: {{ $json.summary.avgTicket }}원
- 운영 매장: {{ $json.summary.storeCount }}개

### 상세 분석
{{ JSON.stringify($json.analysis, null, 2) }}

### 핵심 인사이트
{{ $json.insights.join('\n') }}

### 개선 권고사항
{{ $json.recommendations.join('\n') }}

### 긴급 알림
{{ $json.alerts.join('\n') }}

## 리포트 작성 요구사항

1. **형식**: 텔레그램/카카오톡 메시지에 적합하게 (이모지 활용, 가독성 중시)

2. **구성**:
   - 📊 일일 요약 (3줄 이내)
   - 📈 지점별 성과 (매출순 랭킹)
   - ⚡ 핵심 인사이트 (3개)
   - 🎯 액션 아이템 (우선순위별)
   - 🚨 긴급 알림 (있는 경우만)

3. **톤앤매너**:
   - 전문적이면서도 직관적
   - 숫자는 한눈에 파악 가능하게
   - 결론 먼저, 상세 설명은 뒤에

4. **특별 주의사항**:
   - 전일/전주 대비 10% 이상 변동 시 원인 추정
   - 재고 부족 시 발주 품목 명시
   - 베스트/워스트 지점 언급

리포트를 작성해주세요.
```

## 프롬프트 2: 주간 트렌드 분석 리포트

```
당신은 프랜차이즈 경영 컨설턴트입니다.
아래 주간 데이터를 분석하여 경영진을 위한 전략적 인사이트를 제공해주세요.

## 주간 데이터
기간: {{ $json.period.start }} ~ {{ $json.period.end }}

### 지점별 주간 실적
{{ JSON.stringify($json.weeklyByStore, null, 2) }}

### 요일별 패턴
{{ JSON.stringify($json.dayOfWeekPattern, null, 2) }}

### 메뉴 트렌드
{{ JSON.stringify($json.menuTrends, null, 2) }}

### 운영 지표
{{ JSON.stringify($json.operationMetrics, null, 2) }}

## 분석 요청사항

1. **성과 분석**
   - 지점별 성장률 및 순위 변동
   - 목표 대비 달성률

2. **트렌드 해석**
   - 요일별 매출 패턴의 의미
   - 메뉴 판매 추이 변화 원인

3. **경쟁력 진단**
   - 각 지점의 강점/약점
   - 벤치마킹 포인트

4. **전략적 제언**
   - 단기 개선 과제 (이번 주)
   - 중기 성장 전략 (이번 달)
   - 리스크 요인 및 대응방안

형식:
📋 [주간] 가맹점 경영 분석 리포트

■ Executive Summary (핵심 요약)
■ 지점별 성과 분석
■ 트렌드 & 인사이트
■ 전략적 제언
■ Next Week Action Items
```

## 프롬프트 3: 이상 탐지 알림 (실시간용)

```
당신은 프랜차이즈 본사의 운영 관리자입니다.
아래 이상 징후가 감지되었습니다. 즉시 대응이 필요한지 판단하고 조치사항을 안내해주세요.

## 감지된 이상 징후
{{ JSON.stringify($json.anomalies, null, 2) }}

## 판단 기준
- 매출 급락 (전일 대비 -20% 이상): 🔴 긴급
- 매출 급락 (전일 대비 -10~20%): 🟡 주의
- 재고 부족 (1일 이내 소진 예상): 🔴 긴급
- 폐기율 급증 (5% 이상): 🟡 주의

## 출력 형식

🚨 [긴급/주의] 알림

📍 지점: [지점명]
⚠️ 상황: [이상 징후 설명]
📊 수치: [관련 데이터]
🔍 추정 원인: [가능한 원인 2-3개]
✅ 권장 조치: [즉시 실행 가능한 조치]
📞 에스컬레이션: [필요시 연락처]
```

## 활용 팁

### n8n에서 변수 치환
- `{{ $json.date }}` → 날짜 변수
- `{{ $json.analysis.sales.today }}` → 중첩 객체 접근
- `{{ JSON.stringify($json.data) }}` → 객체를 문자열로

### 토큰 절약을 위한 팁
1. gpt-4o-mini 사용 (비용 대비 성능 우수)
2. 불필요한 상세 데이터는 요약해서 전달
3. 출력 형식을 명확히 지정하여 반복 방지

### 응답 품질 향상
1. 역할(Role)을 명확히 부여
2. 구체적인 출력 형식 제시
3. 예시 포함 시 정확도 향상
