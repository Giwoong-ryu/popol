## 분석 가능 항목

### 1. 매출 분석
- 일별/주별/월별 매출 추이
- 전일/전주/전월 대비 증감률
- 요일별 매출 패턴
- 지점간 매출 비교

### 2. 고객 분석
- 객단가 추이 및 지점 비교
- 재방문율 변화
- 신규 회원 획득률
- 피크타임 집중도

### 3. 메뉴 분석
- 메뉴별 판매 비중 (파이차트용)
- 베스트/워스트 메뉴
- 신메뉴 판매 성과
- 커피/논커피 비율

### 4. 시간대 분석
- 시간대별 매출 분포
- 피크타임 효율성
- 유휴 시간대 파악

### 5. 재고 관리
- 재고 소진 예측 (일평균 사용량 기준)
- 발주 필요 품목 자동 알림
- 재고 회전율

### 6. 운영 효율
- 직원당 매출 (인당 생산성)
- 폐기율 관리
- 날씨/기온과 매출 상관관계

### 7. 경영 인사이트
- 수익성 분석 (매출 대비 비용)
- 성장률 분석
- 개선이 필요한 지점 식별
- 베스트 프랙티스 지점 파악
